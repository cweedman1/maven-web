Incident Investigation Report Example (Nursing Home)

Incident investigation reports are critical for documenting events, identifying root causes, and ensuring regulatory compliance in nursing homes.

This example shows how to structure an investigation report and what information should be included.

What is an Incident Investigation Report?

An incident investigation report documents what happened, why it happened, and what actions will be taken to prevent recurrence.

Key Components of an Investigation Report

  • Description of the incident
  • Timeline of events
  • Staff and resident involvement
  • Root cause analysis
  • Corrective actions
  • Monitoring plan

Example Investigation Report

Incident: Resident fall in hallway.
Findings: Wet floor with no signage.
Root Cause: Environmental hazard and staff oversight.
Correction: Implemented new floor safety protocols and staff training.
Monitoring: Daily environmental checks and weekly audits.

Common Mistakes

  • Incomplete documentation
  • Lack of root cause analysis
  • Failure to define corrective actions
  • No monitoring plan
